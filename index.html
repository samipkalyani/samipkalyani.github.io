Megahunt

The Intelligent-document retrieval system is a powerful tool designed to help users quickly and efficiently retrieve relevant documents from various sources. This system allows users to select directories, SQL and Mongo databases that contain files of type .docx, .ppt, .pdf and xlsx, and then retrieves documents based on user-entered queries. It tags documents with user-provided and self-generated tags to provide tag-based search capabilities, allowing users to easily filter through large numbers of documents. Additionally, the system generates abstractive summaries of these documents, providing users with a quick overview of the key points. Finally, the system opens documents in their native application, allowing users to easily review and edit them.

Bol.io

Our project aims to bridge the communication gap between specially-abled and non-sign language users by providing a two-way sign language communication system. The system uses a GAN model trained on skeletal sequences of Indian Sign Language word actions to generate a video of a non-sign language user performing a sign action based on their audio input. This video is then shown to a specially-abled user connected on the same call using WebRTC. The specially-abled user responds in sign language, and the system recognizes the sign action and generates audio for the entire sequence for the non-sign language user. The end result is a seamless two-way communication system that allows specially-abled users to communicate in their preferred mode while also enabling non-sign language users to understand and respond to sign language. With this project, we hope to make sign language more accessible and inclusive for all.

Research

Generation of Indian sign language by sentence processing and generative adversarial networks.

This paper proposes a method for generating Indian Sign Language videos from sentences using Generative Adversarial Networks (GANs). The method involves converting sentences to glosses and mapping them to a skeletal representation used as input for the GAN model, which outputs corresponding video frames. The proposed approach is evaluated using the SSIM index and a side-by-side comparison of real and generated video frames.
